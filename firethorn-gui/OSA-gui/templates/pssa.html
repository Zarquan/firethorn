The Personal version of the SuperCOSMOS Science Archive

The Personal version of the SuperCOSMOS Science Archive (PSSA) contains an extract of the SSA covering the area (184 < RA/deg < 186, -1.25 < Dec/deg < 1.25) and including a CrossNeighbour table with the Sloan Digital Sky Survey (SDSS) Early Data Release (EDR), which records all EDR sources within 10 arcsec of each PSSA source, and which facilitates spatial matching between the two catalogues.

Querying the PSSA

The PSSA can be queried online through the same SQL interface as the full SSA, and it is recommended that users run queries on the PSSA while developing/debugging them: the PSSA is roughly 0.1% the size of the full SSA, so mistakes are revealed much faster if it is used.

Downloading the PSSA

A .zip file containing the PSSA and instructions for loading it into a SQL Server installation can be downloaded from here. The .zip file is about 50MB in size, and the installed PSSA database takes up about 150MB. The database can be installed via the running of a single .bat file by a user with administrator rights and with SQL Server installed in its default location. To facilitate that automatic installation, please save the .zip file to the top level directory on your C: drive. Instructions are also included in the .zip file for manual installation.
