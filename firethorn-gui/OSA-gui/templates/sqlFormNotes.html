Information on queries

Input may be a query selecting any columns from the selected Databases.

Launching query results in table viewer

One a query has been run and results have been retrieved, you may launch the query results in the "table viewer" which is a tool that allows the creation of graphs based on the query data

Additional options of the query tool

There several features in the query tool, allowing you to:
- Save the data in different formats
- Launch the currently viewed data via SAMP
- Filter the table by key search, rearrange the columns and sort by any column
- View metadata for the selected database
- Select multiple databases and run cross matching queries between them
- Use auto-complete via the use of Ctrl + space when entering a query
