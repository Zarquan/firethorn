OSA_PS_Views.sql

This script creates the required indices for release databases and curation purposes. The script is coded as a set of T-SQL batches so this script can be manually played on the load server through Query Analyser. Additionally, curation application codes can read this script, parse it for the indices created and use this information to enable individual index drop/create calls as well as a complete index creation run (i.e. CU18).
