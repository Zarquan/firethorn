IRAS_Tables.sql

Database schema file for the IRAS Point Source Catalog held in the WFCAM Science Archive. Contains sql scripts to create the relevant tables.
iras_asc Contains the parameters provided for each source in the IRAS associated PSCs.
iras_assoc_cats Contains the parameters provided for the IRAS associated catalogues.
iras_psc Contains the parameters provided for each source in the IRAS PSC.
