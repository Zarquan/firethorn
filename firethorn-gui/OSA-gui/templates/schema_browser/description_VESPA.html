The VESPA Archive is the main repository for VESPA runs on SDSS spectra.

The database is currently split into seven tables:

  • GalProp includes results relative to the galaxy as a whole,
  • BinProp has results which refer to each age bin individually, in the original resolution as recovered by VESPA,
  • HRBinProp splits the recovered solutions into the highest possible resolution (16 age bins), using the appropriate weights,
  • DustProp holds dust information,
  • BinID identifies each VESPA low- or high-resolution bin,
  • RunProp holds run-specific details, such as SSP and dust models used,
  • LookupTable links VESPA's unique identifier with SDSS's own identifiers, such as specObjID, plate, MJD and fiber ID.
Galaxies are identified by a unique index, which can be associated with SDSS's specObjID via the table LookupTable.
