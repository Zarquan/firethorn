The SuperCOSMOS Science Archive is an implementation of the legacy SuperCOSMOS Sky Survey flat-file data in Windows/SQL Server as opposed to the flat-file system accessible via the SSS web interface. It consists of data from digitised sky survey plates taken with the UK Schmidt telescope (UKST), the ESO Schmidt, and the Palomar Schmidt.
The data is organised in schemas, which include tables, views, and constraints.
