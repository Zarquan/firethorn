Database schema for the WISE Preliminary Source Catalog held in the VDFS Science Archive. For further information concerning WISE catalogue tables, please consult the NASA/IPAC Infrared Science Archive WISE webpages.
wise_allskysc Contains the parameters provided for each source in the WISE allsky catalogue.
wise_prelimsc Contains the parameters provided for each source in the WISE preliminary catalogue.
