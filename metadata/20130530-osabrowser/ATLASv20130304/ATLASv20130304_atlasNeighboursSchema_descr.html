Schema file for ATLAS neighbour and cross-neighbour tables NOTE: This schema is not part of the CreateTables set-up for the archive since the table definitions herein are created on-the-fly by the curation procedure (CU16). This file is parsed by CU16 to create the neighbour tables, and also for the web html documentation. For every entry in the table RequiredNeighbours, there should be a parsable table schema in this file.
