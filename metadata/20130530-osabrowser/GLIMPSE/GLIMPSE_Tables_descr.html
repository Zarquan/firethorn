GLIMPSE_Tables.sql

Database schema file for the GLIMPSE Highly Reliable Catalog and the GLIMPSE More Complete Archive held in the WFCAM Science Archive. Contains sql scripts to create the relevant tables.
glimpse1_hrc Contains the parameters provided for each source in the GLIMPSEI Highly Reliable Catalog.
glimpse1_mca Contains the parameters provided for each source in the GLIMPSEI More Complete Archive.
glimpse2_hrc Contains the parameters provided for each source in the GLIMPSEII Highly Reliable Catalog.
glimpse2_mca Contains the parameters provided for each source in the GLIMPSEII More Complete Archive.
glimpse_hrc_inter Contains the parameters provided for each source in the GLIMPSE Highly Reliable Catalog (intermediate).
glimpse_mca_inter Contains the parameters provided for each source in the GLIMPSE More Complete Archive (intermediate).
