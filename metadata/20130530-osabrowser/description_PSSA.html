The Personal version of the SuperCOSMOS Science Archive (PSSA) contains an extract of the SSA covering the area (184 < RA/deg < 186, -1.25 < Dec/deg < 1.25) and including a CrossNeighbour table with the Sloan Digital Sky Survey (SDSS) Early Data Release (EDR), which records all EDR sources within 10 arcsec of each PSSA source, and which facilitates spatial matching between the two catalogues.
The data is organised in schemas, which include tables, views, and constraints.
