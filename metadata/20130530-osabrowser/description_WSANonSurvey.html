Non-Survey

The WFCAM Science Archive contains calibrated data from the UKIRT Wide Field Camera (WFCAM) of both pre-planned public survey operations (through the UKIDSS project), and of competitive smaller non-Survey allocations of time (through the PATT system).

Data from each non-Survey programmes are held in a separate database. The table schemas for the main object catalogues for a given non-Survey programme can be examined using the links on the left. A given non-Survey database will contain a detection table and sometimes a source and mergeLog table (see Data Overview section for more details).
