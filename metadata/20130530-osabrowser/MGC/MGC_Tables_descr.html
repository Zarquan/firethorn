MGC_Tables.sql

Database schema file for the Millennium Galaxy Catalogue held in the WFCAM Science Archive. The original catalogues can be found at http://www.eso.org/~jliske/mgc/. The 3 tables include data from mgc_bright, mgc_faint, mgc_zcat and mgc_gim2d. mgcDetection includes the 32 basic detection parameters - all the columns in mgc_faint and the 1st 32 in mgc_bright. mgcBrightSpec includes the remaining columns in mgc_bright and the data from mgc_zcat. mgcGalaxyStruct contains the data from mgc_gim2d. Repetitions of data columns that made it easier to read the ASCII tables have been removed. Columns have been renamed where there would have been a naming conflict. Contains sql scripts to create the relevant tables.
mgcBrightSpec Combination of extraction parameters that are unique to MGC_BRIGHT and the MGC redshift catalogue.
mgcDetection Table of main extraction parameters for all sources in MGC_BRIGHT and MGC_FAINT http://www.eso.org/~jliske/mgc/
mgcGalaxyStruct Galaxy absolute and structural parameters for 10095 B<20 galaxies in the MGC measured using GIM2D (Simard et al. 2002, ApJS, 142, 1
