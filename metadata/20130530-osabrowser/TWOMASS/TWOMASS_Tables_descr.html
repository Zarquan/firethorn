TWOMASS_Tables.sql

Database schema file for the TWOMASS Source Catalogues held in the WFCAM Science Archive. Contains sql scripts to create the relevant tables.
twomass_psc Contains the parameters provided for each source in the TWOMASS All Sky Release PSC.
twomass_scn Contains the parameters in the Scan Information Table.
twomass_sixx2_psc Contains the parameters provided for each source in the TWOMASS Long Exposure (6x) Scan PSC.
twomass_sixx2_scn Contains the parameters in the Scan Information Table.
twomass_sixx2_xsc Contains the parameters provided for each source in the TWOMASS Long Exposure (6x) Scan XSC.
twomass_xsc Contains the parameters provided for each source in the TWOMASS All Sky Release XSC.
