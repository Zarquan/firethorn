TABLE  twomass_psc

Contains the parameters provided for each source in the TWOMASS All Sky Release PSC.

This table contains brief descriptions of the parameters provided for
each source in the TWOMASS All Sky Release PSC. The table is organized
according to the broad function and utility of the parameters: positional
information, photometric information, and source quality information.
WFAU gratefully acknowledges help from IPAC at Caltech in enabling a
bulk download of the entire catalogue for local deployment in the VSA/WSA.
Any issues encountered or questions raised in use of this incarnation of
the IPAC IRSA catalogue should be communicated to the vsa- or wsa-support helpdesks
in the first instance. For a complete description please see the
user's guide to the 2MASS PSC all-sky data release


Required constraints:
  • Primary key is (pts_key)
  • (ext_key) references twomass_xsc(ext_key)
  • (scan_key) references twomass_scn(scan_key)
