TABLE  atlasSourceXDR7PhotoObj

Cross-neighbours between atlas and PhotoObj.

All SDSS-DR7 sources within 10.0 arcsec of each source from
ATLAS data are recorded in this cross-neighbour table. The table
atlasSource was joined to the PhotoObj table to create these
cross-neighbours. Use this table for any cross-querying of ATLAS
data with sources from SDSS-DR7.
