TABLE  ukirtFSstars

Contains UKIRT Faint Standards observed for calibration observations.

This table contains UKIRT Faint Standard (FS) stars mainly selected from
Hawarden et al. (2001, MNRAS, 325, 563) but supplemented with some
additional stars and with updated photometry as measured with UFTI.
