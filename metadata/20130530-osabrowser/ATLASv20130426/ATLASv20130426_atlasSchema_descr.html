OSA_atlasSchema.sql

Database schema file for ATLAS Survey (ATLAS) catalogue data in the OmegaCam Science Archive. Contains SQL scripts to create the relevant tables.
atlasDetection Contains the individual ATLAS source detections for each frame.
atlasMergeLog Contains frame set details of merged ATLAS MultiframeDetector images.
atlasSource Contains merged sources from detections in atlasDetection.
