Virtual Box /Debian release
---------------------------

1) basic install  (default)


2) Setup sudo access for VM
> su
> root@osa:/home/osa# sudo visudo
add line :  username ALL=(ALL) ALL 


3) Get IP from Ifconfig
> ifconfig


4) Add port forwarding rule (https://www.cs.jhu.edu/~s/teaching/cs460/2013-fall/project3.html)
